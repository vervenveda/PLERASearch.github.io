## Overview

PLERA Search serves as a central discovery gateway for the Verve N Veda public network.

The landing page is designed as a deep-blue search command center with:

- A luminous cyan and electric-blue interface
- A responsive vector magnifying-glass mark
- Curved light ribbons rendered with SVG
- One unified search field
- Search suggestions across tools and public sites
- A searchable application directory
- A searchable network directory
- Locally stored favorite tools
- No account requirement
- No analytics or tracking scripts

The interface is intentionally lightweight, portable, and independent of external frameworks.

---

## Purpose

PLERA Search is being developed to help visitors:

- Search broadly from one starting point
- Reach trusted reference tools more quickly
- Compare different kinds of resources
- Move among education, research, law, weather, art, civic, and community platforms
- Access specialized insurance and shopping directories
- Discover the broader Verve N Veda ecosystem
- Keep favorite PLERA tools stored locally in their browser
- Use the site without creating an account

PLERA Search is a gateway. It does not replace independent evaluation, source comparison, or professional advice.

## Unified Landing-Page Search

The landing page search field searches two collections at once:

1. PLERA Search applications
2. Connected public landing pages

As a visitor types, the page can:

- Display matching suggestions
- Filter PLERA application cards
- Filter connected public-site cards
- Open a single strong match directly
- Search titles, descriptions, categories, repository names, and filenames

The `/` keyboard shortcut places focus in the main search field.
